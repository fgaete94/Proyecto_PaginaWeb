A continuación se explica el paso a paso para ingresar a la página web:

1. Una vez descomprimido el archivo RAR, posicione en la carpeta "proyecto web".
2. Inicie la terminal utilizando como ruta la ubicación en la que se encuentra actualmente.
3. Cree un entorno virtual si es necesario. En caso de ejecutar en Windows, active el entorno virtual llamado "página"; 
   si ejecuta en terminal Macintosh, inicie el entorno virtual "web".
4. Cambie el directorio para situarse ahora en la carpeta "enfqx". En esta carpeta, deberá visualizar el archivo manage.py.
5. Si en el paso 2 creó un entorno virtual nuevo, ejecute pip install -r requirements.txt.
6. Una vez finalizada la instalación anterior, ejecute el comando python manage.py runserver.
7. Copie la dirección de enlace entregada por la terminal y agregue al final la palabra "index" para ingresar a la página, 
   tal como el ejemplo siguiente: "https://127.0.0.1/index".
8. Navegue por la página.
